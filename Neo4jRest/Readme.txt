Neo4jRest

For running this module you need to have a neo4j server running locally on the default url http://localhost:7474

In res folder you can find a zip "default.graphdb.zip" - unzip this folder and use it as database location when you start the neo4j server. 
This zip contains the configurations for a better performance from neo4j server instance.

Run the project by launching as a Scala Application the class Main from /src/main/scala/com/akka/ws/neo4j/rest
